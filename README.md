# CS525 Final Project 

### How to Run
#### Dependencies
    pip install nltk
    pip install django

- python version 3.6.3

- django version 2.0.0

#### Instruction
- `git clone https://github.com/uxroc/infoweb`

- go to directory  `website`

- Run `python manage.py runserver`

- Open a browser, and enter `https://localhost:8000`

- **Enjoy!**
